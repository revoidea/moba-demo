### 开发网站的步骤
- 在style.scss里面
- 第一步：重置样式（写css要注意的点）
- 第二步：网站色彩和字体定义（colors,text）:
- 思考：
- 1.整体看网站：
- a.主色调是什么？用到几种颜色？
- b.字体大概是什么样子？字体的尺寸，大小，对齐方式？
- 第三步：通用的flex布局定义
- 第四步：常用的边距定义（margin，padding）
